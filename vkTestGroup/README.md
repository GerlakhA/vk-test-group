# Инструкция по запуску проекта

1. Cколонировать репозиторий: 
```bash
git clone https://github.com/GerlakhA/test-filter-group.git && cd test-filter-group
```

2. Установить зависимости:
```bash 
npm i
```

3. Запустить сервер
```bash 
npm run server
```

4. Запустить сайт
```bash 
npm run dev
```
